# bitcoin to bullshit

### description
this chrome extension replaces "bitcoin" with what it really is: "bullshit".

### installation instructions
1. click "clone or download" and then "download zip".
2. unzip the downloaded file.
3. go to [chrome://extensions/](chrome://extensions/) on chrome.
4. check the "developer mode" box.
5. check "load unpacked extension" and select the folder your unzipped files are in.
6. go wild.
